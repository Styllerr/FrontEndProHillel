1. В чём разница между BOM и DOM?
	BOM: позволяет нам через свойства и методы объектов взаимодействовать с ВЕБ-БРОУЗЕРАМИ (или их аналогами)
	DOM: позволяет нам через свойства и методы объектов получить доступ и возможность обновлять содержимое, структуру и стиль ДОКУМЕНТОВ.
 
2. В чём принципиальная разница между выводом сообщений с помощью alert и console.log? 
	Alrt выводит сообщение в модальное окно, а console.log выводит сообщение в консоль браузера.
 
3. Чем отличаются ошибки чтения от ошибок выполнения?
	Ошибки чтения - это синтаксические ошибки в коде, и выявляются они еще до его выполнения. А ошибки выполнения (Runеime Error) - это ошибки выявленные в процессе выполнения кода.
 
4. Всё ли верно? var m; var M; var 7e;
	Переменная не может начинаться с числа. Остальные варианты верны.

6. Какой будет результат выражений? По возможности, объясните почему именно такой результат. 
	"23" + 20 + '' = 2320
	Происходит конкатенация, т.к. одно из слогаемых типа string
 
	"1" + (20 + 3) = 123
	Действия в скобках приоритетны, поэтому сначала сложение 20+3, потом конкатенация строк 1 и 23
 
	"a" / 5 = NaN
	Первым делом "а" преобразуется в Number - получаем NaN. NaN делить на число - получаем NaN
 
	("str" || 5) > 0 || (10 && true) = true
	("str" || 5) - в результате логический оператор или вернет первое true в не приведенном виде, т.е. "str". Он, в свою очередь преводится к типу Number, для сравнения с 0. Получаем NaN > 0 - false
	Результатом (10 && true) будет true,  т.к. оба значения истина, оператор И возвращает нам второе, которое очень кстати типа "Boolean". Теперь имеем логику вида:
	
		false || true . Оператор ИЛИ возвращает значение, соответствующее первому true - true
